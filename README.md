# TFM - Optimización de Enrutado en Almacenes

## Descripción

Este proyecto desarrolla un algoritmo de optimización de rutas para operaciones de enrutado en almacenes.

El objetivo es minimizar la distancia recorrida por los operarios durante la distribución de pedidos, mejorando la eficiencia logística y reduciendo los tiempos de operación.

El sistema permite modelar la distribución del almacén, procesar ubicaciones de productos y generar rutas optimizadas para la recogida de artículos.

---

## Objetivos

- Reducir la distancia total recorrida en vacío.
- Optimizar los tiempos de distribución de pedidos.
- Analizar el impacto de la optimización en la productividad del almacén.
